# storyvansh - a choose-your-own style game, but using ai!

The basic goal is to learn more about APIs, good coding practice, UI, and later some fullstack shit

Will keep updating this as new versions come out

Technical details of versions are in the **CHANGELOG**, this will document more abstract explanations

## v0.0.1
Made a program to help log API calls.

Started with a very basic version of the application. Idea was to have a conversation between two people, the person gemini is larping as and me.
- The starting of the chat had to be based on a theme, so made a list of 5 of them and
  the chosen one is called the **genesis**.
- The character gemini is playing as is referred to as **God** for now, when adding
  the *__-vansh* feature, will change this.

This version of the application is good for development, but the concept of the user being alongside *__-vansh* doesn't make too much sense and complicates the prompts needed, so a better option will be to make it entirely first person and generate the *vansh-variant* in different scenarios.
